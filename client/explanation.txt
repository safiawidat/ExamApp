E-Test System - Project Explanation

Project Assignment 2:
This project is a Full React client-side application for managing online exams.  
The system works without a server and uses mock data to simulate a real system.

Project Goal:
The goal of this project is to create an exam management system for teachers and students.  
Teachers can manage exams, while students can view and submit exams.

Required Client Content:

AUTH:
- Login
- Register

Teacher Pages:
- TeacherDashboard
- AddExam
- ExamList
- ExamDetails
- ScoresPage

Student Pages:
- StudentPortal
- StudentForm

Navigation:
- NavigationMenu for navigation, role display and logout.

Services:
- Configuration Service (configService.js)
- Mock API DB Service (mockDb.js + examService.js)
- Logger Service
- Storage Service
- Notify Service

Main Features:
- Login for teacher and student users.
- Register new users.
- Teacher can create exams.
- Teacher can publish and delete exams.
- Teacher can view student submissions and grades.
- Student can view available exams.
- Student can enter and submit exams.
- Navigation menu with logout option.
- Local storage support for saved users.
- Unit testing using Vitest.

Users:

Teacher:
- Login to the system.
- Create exams.
- View exam list.
- Publish exams.
- Delete exams.
- View student submissions.
- View grades and feedback.

Student:
- Register to the system.
- Login to the system.
- View available exams.
- Enter an exam.
- Submit answers.
